What is UtilitiesPlus?

UtilitiesPlus is a feature-rich Minecraft Spigot plugin designed with practicallity in mind. Its many commands and other various features make it a server necessity for moderation, entertainment, ease-of-use and much more. Its powerful configuration files can also allow you to completely customize how the plugin works.

UtilitiesPlus also has two different modes - Backend, and Proxy. Backend mode will be used by the majority of servers as it only requires common server software such as Bukkit, Spigot or Paper. Proxy however, requires a network running software such as BungeeCord, Waterfall or Velocity. These different modes add and change certain features depending on how the server is setup. For example: When the plugin is setup for a BungeeCord network, banning/kicking a player will kick them directly from the proxy. This means that if you use a plugin that sends you to a fallback server when you get disconnected, players won't be able to bypass bans.

If you are not running a server network, most features are still available to you. The only things you wont get are things that are only supported by having a proxy. (e.g. /swap, which allows you to swap servers on your network).

What makes it so useful?

One of the main things that makes UtilitiesPlus so useful is simplicity. Our easy-to-use plugin design allows for quick and easy setup and use. This means that both players and staff members can quickly do whatever they want, when they want with as little confustion as possible.
